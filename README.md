# pinglog
Bash-скрипт для логирования пингов, который запускает непрерывную серию пингов к указанному IP. Успешные ответы пишутся в один файл, ошибки и таймауты - в другой. Все записи содержат временную метку.\
Скрипт не зависает при потере соединения благодаря таймауту. Параметры можно передавать в любом порядке.\

Остановить скрипт - *Ctrl+C*.\

## Использование

`./pinglog.sh [IP-адрес] [опции]`

Предварительно дать права на выполнение:
`chmod +x pinglog.sh`

## Опции
```
-c, --clear   - очистить логи перед запуском 
--no-log      - отключить запись в файлы (только вывод в консоль)  
-w <файл>     - задать имя файла для записи логов (по умолчанию log.txt)  
--high <ms>   - если время отклика превышает этот порог, запись помечается как ERROR
-h, --help    - показать справку
```

## Примеры
```
./pinglog.sh 8.8.8.8

./pinglog.sh 8.8.8.8 -c -l pings.log -e errors.log

./pinglog.sh 8.8.8.8 --no-log

./pinglog.sh -c 8.8.8.8
```
