# pinglog
Bash-скрипт для логирования пингов.

## Использование

`./pinglog.sh [IP-адрес] [опции]`

## Опции

-c, --clear&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- очистить логи перед запуском  \
--no-log&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- отключить запись в файлы (только вывод в консоль)  \
-l <файл>&nbsp;&nbsp;&nbsp;&nbsp;- задать имя файла для успешных пингов (по умолчанию ping_log.txt)  \
-e <файл>&nbsp;&nbsp;&nbsp;- задать имя файла для ошибок (по умолчанию error_log.txt)  \
-t <сек>&nbsp;&nbsp;&nbsp;&emsp;- таймаут для каждого пинга (по умолчанию 1)  \
-h, --help&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- показать справку

## Примеры
```
./pinglog.sh 8.8.8.8

./pinglog.sh 8.8.8.8 -c -l pings.log -e errors.log

./pinglog.sh 8.8.8.8 --no-log

./pinglog.sh -c 8.8.8.8
```
## Что делает

Скрипт запускает непрерывную серию пингов к указанному IP. Успешные ответы пишутся в один файл, ошибки и таймауты - в другой. Все записи содержат временную метку.\
Скрипт не зависает при потере соединения благодаря таймауту. Параметры можно передавать в любом порядке. Если указан --no-log, опции -l, -e и -c игнорируются.\

Логирование включено по умолчанию. Если нужен только вывод в консоль - используйте --no-log.\

Остановить скрипт - *Ctrl+C*.\

## Формат логов

Успех:
[2026-07-01 14:23:45] ICMP_SEQ=1 TIME=12.3 ms

Ошибка:
[2026-07-01 14:23:48] ERROR: ICMP_SEQ=4 - Request timeout (no response)
